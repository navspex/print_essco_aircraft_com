# ESSCO Aircraft — Print-On-Demand

Aviation manual printing services. Upload your PDF, get instant pricing.

**Live site:** https://print.esscoaircraft.com

## Stack
- React 18 + TypeScript
- Vite 5
- Tailwind CSS 3
- Cloudflare Pages + R2 Storage
- Shopify Draft Orders API

## Routes
- `/` — Manual printing calculator (landing page)
- `/posters` — Poster printing info
- `/posters/calculator` — Poster pricing calculator
